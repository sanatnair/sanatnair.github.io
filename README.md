# Sanat Nair — GitHub Pages Starter

This repository hosts my personal portfolio using GitHub Pages.

## Quick Start
1. Create a public repo named `sanatnair.github.io` on GitHub.
2. Upload these files to the root of that repo.
3. Go to **Settings → Pages** and ensure Source = `main` branch.
4. After 1–2 minutes, your site will be live at: https://sanatnair.github.io

## Customize
- Replace `Resume_SN.pdf` with your real résumé PDF (same filename).
- Swap placeholder images in `assets/images/`.
- Edit project pages in `/projects`.
